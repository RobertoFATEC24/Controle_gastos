# Controle_gastos
Conteúdo para p2 de 03/06

## Tecnologias Utilizadas

* React Native
* Expo
* JavaScript
* React Navigation
* SQLite
* StyleSheet

## Requisitos

* Node.js
* npm
* Expo CLI (opcional)
* Expo Go (Android ou iOS)

## Instalação

1. Clone o repositório:

git clone URL_DO_REPOSITORIO

2. Acesse a pasta do projeto:

cd Controle_Gastos-p2

3. Instale as dependências:

npm install

4. Instale as bibliotecas utilizadas:

npm install @react-navigation/native

npm install @react-navigation/native-stack

npm install react-native-screens

npm install react-native-safe-area-context

npm install @react-native-picker/picker

npx expo install expo-sqlite

npm install @react-native-community/datetimepicker

## Execução

Para iniciar o projeto execute:

npx expo start

ou

npx expo start --clear

Após a execução será exibido um QR Code.

Abra o aplicativo Expo Go no celular e escaneie o QR Code para executar a aplicação.
